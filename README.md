# Lingo — 나만의 AI 영어 튜터 v2

개인 사용을 위한 **설치형 웹앱**. 안드로이드 홈 화면에 추가해 앱처럼 쓸 수 있어요.
**1개 HTML 파일 + 서비스 워커 1개**로 구성된 가벼운 단일-페이지 앱.

```
lingo/
├── index.html   ← 앱 본체
├── sw.js        ← 오프라인 캐시
└── README.md    ← 이 파일
```

## ✨ v2 핵심 기능

### 1. 매일 출석 시스템 (게임화)
- 홈 상단 **월간 출석 캘린더** — 출석한 날은 황금 칸
- 출석할 때마다 **+5 XP · +1 💎(보석)**
- 7일·14일·30일 마일스톤에 보너스 보석 / 하트 풀충전
- 보석은 상단바 💎 카운터에 누적

### 2. 단어 매칭 게임 (9개 스테이지)
- 한국어 ↔ 영어 카드 짝 맞추기 (메모리 게임)
- 8장 → 12장 → 16장 → 20장으로 **단계별 난이도 상승**
- 시간/시도 횟수에 따라 **별 1~3개** 획득
- 별 3개 만점 시 보석 보너스
- 스테이지 클리어해야 다음 잠금 해제

### 3. 단계별 문법 학습 (25개+)
- CEFR **A1 → A2 → B1 → B2** 단계로 구조화
- 각 문법마다:
  - 📖 **뜻 / 개념** — 한국어로 쉽게
  - 🧩 **어순 / 구조** — 시각적 공식
  - 🔍 **자세한 규칙** — 변형·예외·주의점
  - 💬 **예문 3개** — 한·영 짝
  - 📝 **3문제 퀴즈** — 통과 시 ✅ 마스터 + 15 XP + 1 💎
- 진행률 자동 추적

**커버하는 문법:** be 동사, 일반동사 현재형, 부정/의문, 관사, 복수형, There is/are, 대명사·소유격, 과거형, 현재진행, 미래(will/be going to), 비교급/최상급, can, 의문사, 현재완료, 관계대명사, 조동사, 수동태, 가정법 1·2·3형, to부정사/동명사, 분사구문

### 4. 미디어 학습 — 드라마/영화/노래/동화
- **📺 드라마**: Friends, Modern Family, Extra English, Doctor Who, The Crown
- **🎬 영화**: Finding Nemo, Toy Story, Pursuit of Happyness, Forrest Gump, Harry Potter
- **🎵 노래**: Twinkle Twinkle, Let It Be, Imagine, Shape of You, Someone Like You + LyricsTraining
- **📖 동화**: 이솝우화/안데르센/그림형제 — 일부는 본문 직접 내장(단어 탭 시 사전)
- 각 항목별 **핵심 표현 · 가사/본문 · 외부 영상 링크** 제공

### 5. 랭크 시스템
🐣 병아리 → 🐦 참새 → 🕊️ 비둘기 → 🦅 독수리 → 🐲 용 → 🔥 불사조 → 🌟 신화
(XP 기반, 프로필에 진행 바)

### 6. 레슨 별점
- 정확도 ≥95% → ⭐⭐⭐ (보석 보너스)
- 정확도 ≥75% → ⭐⭐
- 그 외 → ⭐

## 📱 안드로이드 설치

### 가장 빠른 경로 (file://)
1. `index.html` + `sw.js` 폰으로 함께 전송
2. 폰 크롬에서 `index.html` 열기
3. **⋮ → 홈 화면에 추가**

⚠️ `file://`에서는 서비스 워커가 등록되지 않고, 음성인식·일부 API가 막힐 수 있어요.

### 추천: HTTPS 호스팅 (전 기능 동작)
무료 옵션:
- **GitHub Pages**: 리포지토리 푸시 → Settings → Pages 활성화
- **Vercel/Netlify**: 폴더 드래그앤드롭
- **Cloudflare Pages**: 한국 속도 빠름

HTTPS면 자동으로 **오프라인 동작**(sw.js)도 켜집니다.

## 🛠 확장 가이드

```
index.html
├── DEFAULT_STATE          — attendance, wordGame, grammarProgress 포함
├── UNITS[]                — 5개 A1 단위 (vocab 추가 가능)
├── ASSESSMENT_BANK[]      — CEFR 진단 12문항
├── CURATED_CLASSICS[]     — Project Gutenberg 6편
├── READING_TOPICS[]       — AI 글 주제 10개
├── RESOURCES[]            — 자료 허브 10 카테고리
├── GRAMMAR_LESSONS[]      — 문법 25+개 (확장 쉬움)
├── WORD_GAME_STAGES[]     — 게임 스테이지 9개
├── MEDIA_RESOURCES        — drama/movie/song/fairytale 각 5~7개
└── RANKS[]                — 7단계 랭크
```

추가하기 쉬운 곳:
- 새 문법 → `GRAMMAR_LESSONS` 에 push (id, level, title, concept, structure, detail, examples, quiz)
- 새 게임 스테이지 → `WORD_GAME_STAGES` 에 push
- 새 드라마/영화/노래/동화 → `MEDIA_RESOURCES[탭]` 에 push
- 새 단어 → `UNITS[].vocab` 에 push
- 새 랭크 → `RANKS` 에 push (min XP 기준 정렬)
